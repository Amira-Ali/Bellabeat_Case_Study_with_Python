# Bellabeat Smart Device Usage Analysis

Analysis of Fitbit smart device data to uncover consumer behavior patterns and guide Bellabeat's marketing strategy. Using Python and Excel, explores activity levels, sleep, steps, and calorie trends across 24 users, delivering actionable recommendations for the Bellabeat app.

## 📊 Key Findings
- Steps and calories are positively correlated — the more you walk, the more you burn.
- Sleep duration has no measurable effect on activity level or distance travelled.
- Saturdays are among the most active days — users exercise more on weekends.
- Peak active hours fall between 12pm and 7pm.
- 91.6% of users prefer light-intensity activity.

## 💡 Recommendations
- Send app notifications and promotions during peak active hours and weekends.
- Suggest personalised weekend plans: diet, walking routes, and navigation.
- Offer gentle exercise suggestions and relaxation content for light-activity users.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, NumPy)
- Microsoft Excel
